# Azure Cloud Foundations Lab

##  Overview
This project provides a hands-on introduction to cloud computing using Microsoft Azure. It focuses on setting up a cloud environment, understanding core cloud concepts, and deploying basic resources within the Azure Free Tier.

The goal is to bridge the gap between theoretical knowledge and practical cloud experience, particularly for aspiring DevOps and cloud engineers.

---

##  Project Objectives
- Set up a Microsoft Azure Free Tier account
- Navigate and explore the Azure Portal
- Understand cloud governance (Subscriptions & Resource Groups)
- Learn Identity and Access Management (RBAC basics)
- Deploy a basic cloud resource (Virtual Machine or Storage)
- Explore cost management and budgeting tools
- Understand the Shared Responsibility Model

---

##  Tasks Completed

### 1. Account Setup
- Created and verified an Azure Free Tier account
- Completed identity and payment verification

### 2. Azure Portal Exploration
- Navigated dashboard and service menu
- Explored key services like Virtual Machines and Storage Accounts

### 3. Governance Setup
- Identified active Subscription
- Created Resource Group:
  - **Name:** `rg-devops-lab`
  - **Region:** West Europe

### 4. Identity & Access Management
- Explored Azure Active Directory (Azure AD)
- Reviewed Role-Based Access Control (RBAC)
  - Owner
  - Contributor
  - Reader

### 5. Region Selection
- Selected **West Europe** for optimal latency and availability

### 6. Resource Deployment
- Deployed a basic resource:
  - Virtual Machine (Ubuntu Linux) *(or Storage Account if applicable)*

### 7. Cost Management
- Accessed Cost Management + Billing
- Reviewed Cost Analysis tools
- Configured budget monitoring (optional)

---

##  Region Used
**West Europe**

**Reason:**
- Closest Azure region to Nigeria
- Lower latency and improved performance
- High availability and reliability

---

##  Key Learnings
- Structure and organization of Azure resources
- Importance of Resource Groups in cloud management
- Basics of Role-Based Access Control (RBAC)
- Differences between IaaS, PaaS, and SaaS
- Cloud cost monitoring and budgeting practices
- Real-world application of the Shared Responsibility Model
