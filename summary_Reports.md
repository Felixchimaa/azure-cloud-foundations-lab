##  Region Selection

The selected region for this project is **West Europe**.

### Reason for Selection:
- It is geographically closer to Nigeria compared to other Azure regions
- Provides lower latency and faster response times
- Offers high availability and reliability for deployed resources

Choosing an appropriate region is important in cloud computing as it directly affects performance, cost, and user experience.

---

##  Shared Responsibility Model

The Shared Responsibility Model defines how security responsibilities are divided between the cloud provider (Azure) and the user.

### Azure Responsibilities:
- Physical data centers and infrastructure
- Hardware and networking
- Power, cooling, and physical security

### User Responsibilities:
- Operating system updates and patches (for Virtual Machines)
- Application security
- Identity and access management
- Data protection and backups

### Example from This Project:
For the deployed Virtual Machine:
- Azure is responsible for maintaining the physical servers and infrastructure
- I am responsible for securing the OS, managing access (SSH/login), and protecting any applications running on the VM
